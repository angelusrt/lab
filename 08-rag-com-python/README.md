# RAG com Python

Estudando o uso de LLM com Python para a construção de fluxos.

## Build

Instalando...

```{bash}
python3 -m venv .venv
source .venv/bin/activate

pip install google-generativeai numpy
```

Você precisará definir o token do Gemini na variável de ambiente.

```{env}
GOOGLE_API_KEY=[token]
```
